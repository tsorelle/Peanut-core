# Committees and Appointments
## Making Changes

Here is a simple example of how to make changes to committees and appointments.  These screenshots are from SCMY.org, 
but the same feature in austinquakes.org is very similar. We bigin on the 'Committees & Appointments' page:

SCYM.ORG: Directory > Appointments & Committees<br>
AustinQuakers.org: Community > Directory > Committees & Appointments

![committees-main.jpg](img/committees-main.jpg)

Select a Committee of appointment position from the drop-down list.  In this example we will select 'Yearly Meeting Clerk'.

![select-clerk.jpg](img/select-clerk.jpg)

![committees-clerk-screen.jpg](img/committees-clerk-screen.jpg)

Since the Clerk is a single member postion, only one person is listed.  If you select a committee, such as 'Technology Committee', you will see a list of all the members.

![committees-tech-screen.jpg](img/committees-tech-screen.jpg)

In 2026 Kevin King was appointed and approved as Yearly Meeting Clerk, replacing the previous Clerk, Richard Corl.
This is how we make that change in the system. First we will click 'Add new member' and add Kevin as clerk.

![committees-new-member.jpg](img/committees-new-member.jpg)

Search and select 'Kevin King'.

![committees-select-member-1.jpg](img/committees-select-member-1.jpg)

The member input accepts these fields:
- Status (Nominated, First Reading, Approved, Withdrawn, )
- Start of Service - the year the appointment begins.
- End of Service - the end date of the term of service.
- Relieved - the actual date the person left the position. It is typically different from "End of Service", because plans may change.

Click 'Save Changes' when ready.

![committees-new-appointment.jpg](img/committees-new-appointment.jpg)

Now both Kevin King and Richard Corl are listed as Clerk, so we need to remove Richards name as a current appointee. 
First click on Richard Corl's name.  And in the member form enter the date that he was relieved from the clerkship, 
and click 'Save Changes'.

![committees-relieved-date.jpg](img/committees-relieved-date.jpg)

Now only Kevin King is listed as the current clerk and our task is complete.

If you want to see a list of previous appointments, click on the drop down list. And select 'Current and Former appointees'.

![committees-select-former.jpg](img/committees-select-former.jpg)

Now you can see a list of previous appointments.

![committees-former-clerks.jpg](img/committees-former-clerks.jpg)
