# Message Content Editing

Peanut uses the TinyMCE editor to edit content for mailings and other tasks. Our extenstion to the editor allows for 
saving content and retrieving content with versioning.

Here is an example of how to use the editor from the Mailing form as featured in Scym.org and Austinquakers.org.

Above the editing field at the right hand area of the tool bar you will see a button labeled "Save". 
Clicking this button will save the current content of the editor. It will disabled and "greyed out" until you add some
new content or make changes.

![editor-1-save-button.jpg](img/editor-1-save-button.jpg)

When you click the "Save" button, the editor will save the current content of the editor. You will see this dialog:

![editor-5-savechanges.jpg](img/editor-5-savechanges.jpg)

If you check ‘Shared’ your text will be available to others who use the form.  Otherwise it is just for you. 
Now, when you want to open the same content in future, click ‘File’ on the menu, and select open.

![editor-2-open-file.jpg](img/editor-2-open-file.jpg)

Then you get a list of previously saved content that you can select from.

![editor-3-getcontent.jpg](img/editor-3-getcontent.jpg)

Content will be loaded into the editor as in this example:

![editor-4-loadedcontent.jpg](img/editor-4-loadedcontent.jpg)

You can also choose ‘New Document’ which clears the text, or ‘Save as’ to save your content under another name.

After you load some previous content, whenever you click ‘Save’ it creates a version.  To reload a previous version, click ‘File > Open”.  
If multiple versions are available, you’ll get a “Versions” tab that allows you to select a previously saved version.

![editor-6-versions.png](img/editor-6-versions.png)
